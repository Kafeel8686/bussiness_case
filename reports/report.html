BTCUSDT Data Quality Report

Description of Steps Taken

Data Download

Downloaded historical 1-minute Kline data for BTCUSDT from Binance starting from January 2017 to June 2024 using asynchronous requests, ensuring data is not re-downloaded if it already exists.

Data Preprocessing

Unzipped and read CSV files, combined them into a single DataFrame, and assigned appropriate column names.

Data Quality Evaluation

Evaluated the dataset for completeness, missing values, duplicates, data types, logical consistency, and anomalies.

Anomaly Detection

Identified unusual spikes and drops in price and volume using statistical methods and moving averages.

Report Generation

Generated an HTML report summarizing the findings, including charts and observations.
